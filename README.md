# NEXUS Portfolio

Futuristic developer portfolio SPA with a cyberpunk UI, animated ambience, real contact workflow, an AI assistant, and an upcoming-project teaser video card.

## ✦ Features

- **SPA navigation** with view routing: Home / About / Thank You / Admin
- **Cyberpunk UI system**: glitch text, scanlines, neon glows, hover-reactive cards
- **Animated background ambience**: particles + starfield + perspective grid
- **Theme controller**: light/dark mode + multiple color palettes
- **Projects carousel**: category tabs + swipe/scroll + left/right nav buttons
- **Video project support**:
	- Autoplay preview (muted, loop, playsInline)
	- Visible **UNMUTE / MUTE** button (audio starts only after user click)
- **“COMING SOON” highlighting** for under-development projects
- **AI chatbot assistant** with streaming responses (Groq OpenAI-compatible endpoint)
- **Contact form** via Formspree with success/error overlay and auto-redirect
- **Thank You page** with a mini-game (best score stored in localStorage)
- **Admin Command Center** to edit projects/skills (demo/local usage)
- **Local data persistence** with version-based invalidation (prevents stale cached data)

---

## ✦ Tech Stack

- **Frontend:** React + TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Icons:** lucide-react (+ brand icons via react-icons where needed)
- **AI Chat:** Groq (OpenAI-compatible streaming)
- **Contact:** Formspree


---

## ✦ Admin Command Center

Admin view exists for demo/local management of projects and skills.

- Access is via a hidden trigger in the footer (small dot).
